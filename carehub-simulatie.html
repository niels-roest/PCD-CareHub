Modelaannames per scenario

Elk scenario is cumulatief — D bevat alle effecten van A, B en C.

A

Growtivity

  • + +4% behandelcapaciteit (admin −2u/week, no-shows −30%, betere matching)
  • + −1% vraag (hogere intakekwaliteit, minder verkeerde plaatsingen)
  • → Start maand 3, volledig effect na 12 maanden
B

Syntillio (bovenop A)

  • + +2% extra capaciteit (kortere trajecten via remote monitoring)
  • + −25% vraag via AI-triage (instelbaar)
  • → Start maand 3, volledig effect na 15 maanden
C

Overheidssteun (bovenop B)

  • + +1.5% extra capaciteit (IZA-transformatiegeld, meer organisaties)
  • + −0.5% extra vraag (preventief beleid AZWA 2025)
  • → Start maand 3, volledig effect na 12 maanden
D

Verzekeraarssteun (bovenop C)

  • + +0.7% extra capaciteit (VGZ Voorop in Zorg €60M)
  • + −0.8% extra vraag (preventieve zorg gefinancierd door verzekeraars)
  • → Start maand 1, volledig effect na 18 maanden
